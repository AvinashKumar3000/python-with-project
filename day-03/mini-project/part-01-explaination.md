## what is CSV

- CSV - comma separated value.
- We know excel sheet, which stores data in rows and columns. 
- As simple way to keep data in row & cols is csv.
- Each line represents row.
- Each data separated with comma is column values.

example

```csv
id,name,age
12,sample,23
13,Ravi,20
14,Anita,22
15,Karan,24
```
